Optimasi Pelayanan Kesehatan dengan Simulasi Antrean

Aplikasi simulasi berbasis Discrete Event Simulation (DES) untuk menganalisis dan mengoptimalkan sistem antrean di fasilitas kesehatan menggunakan framework SimPy.

Simulasi Realistis

Parameter Interaktif

Atur kapasitas dokter, waktu kedatangan, dan durasi layanan secara real-time untuk melihat dampaknya.

Rekomendasi AI

Sistem memberikan rekomendasi optimasi berdasarkan hasil simulasi untuk meningkatkan efisiensi layanan.

Visualisasi Data

Grafik dan visualisasi interaktif untuk memahami pola antrean dan kinerja sistem secara intuitif.

Data Riil

Integrasi dengan data riil dari database klinik untuk simulasi yang akurat dan relevan.

Mobile Friendly

Aplikasi dapat diakses dari berbagai perangkat dengan tampilan yang responsif dan user-friendly.
